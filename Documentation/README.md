# 🚀 CONTROL_NEXUS — Laravel Portfolio CMS

Thank you for purchasing CONTROL_NEXUS.

A modern portfolio CMS built with Laravel 12.

---

## 📌 Features

* Admin Dashboard
* Portfolio Management
* Homepage Builder
* GitHub Integration
* Image Gallery

---

## ⚙️ Installation

```bash
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate --seed
php artisan storage:link
```
