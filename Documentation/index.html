🚀 CONTROL_NEXUS Documentation

A modern Portfolio CMS built with Laravel 12.

📌 1. Server Requirements

  • PHP 8.2+
  • MySQL 8+
  • Composer
  • Node.js (optional)

⚙️ 2. Installation

Step 1: Upload Files

Upload project files to server.

Step 2: Install Dependencies

composer install

🔧 3. Environment Setup

cp .env.example .env
php artisan key:generate

🗄️ 4. Database Setup

DB_DATABASE=control_nexus
DB_USERNAME=root
DB_PASSWORD=

🚀 5. Migration & Seeder

php artisan migrate --seed
This creates demo admin & portfolio data

📁 6. Storage Setup

php artisan storage:link

🎨 7. Assets (Optional)

npm install
npm run build
