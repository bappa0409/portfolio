# 🚀 Developer-focused Portfolio CMS with GitHub Integration & Dynamic Homepage Builder

A modern and dynamic portfolio management system built with Laravel. This system allows developers, freelancers, and agencies to showcase their work, manage projects, and integrate GitHub repositories seamlessly.

---

## ✨ Features

* 🔐 Admin Dashboard
* 🧑‍💼 Manage Portfolio Projects
* 🖼️ Image Upload & Gallery
* 🔗 GitHub Repository Integration
* 📊 Visitor Tracking
* 📱 Fully Responsive Design
* ⚡ Clean and Optimized Code
* 🛠 Built with Laravel (MVC Architecture)

---

## 🚀 Installation Guide

1. Extract the ZIP file
2. Copy `.env.example` to `.env`
3. Update database credentials in `.env`
4. Run the following commands:

```
composer install
php artisan key:generate
php artisan migrate --seed
npm install && npm run build
```

5. Start the server:

```
php artisan serve
```
